# MLOps Engineering Internship - Mini Pipeline

## Overview
This repository implements a small MLOps-style batch pipeline:
- Loads `data.csv` (cryptocurrency OHLCV)
- Computes a rolling mean on the `close` column
- Generates binary signals (1 if close > rolling mean else 0)
- Emits machine-readable metrics (`metrics.json`)
- Writes detailed run logs (`run.log`)
- Is containerized with Docker

## Files
- `run.py` - Main script (CLI)
- `config.yaml` - Configuration (seed, window, version)
- `data.csv` - Input dataset (provided)
- `requirements.txt` - Python dependencies
- `Dockerfile` - Build image
- `metrics.json` - Example output (generated by run)
- `run.log` - Example log (generated by run)
